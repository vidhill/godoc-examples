# godoc-examples

Links

- Go official blog: https://go.dev/blog/examples
- Go docs: https://pkg.go.dev/testing#hdr-Examples
- https://pkg.go.dev/golang.org/x/tools/cmd/godoc

### Install `godoc`

```bash
$ go install golang.org/x/tools/cmd/godoc@latest
```

### Run

```bash
$ godoc -http=:6060
```

### Run tests

```bash
$ go test ./...
```
